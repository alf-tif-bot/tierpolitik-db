# Review / Freigabe

## Ziel
Schneller Qualitätscheck vor Veröffentlichung oder Übergabe.

## Input
Entwürfe aus den Production-Schritten.

## Output
Status: `freigeben`, `nachschärfen`, `verwerfen` + kurzer Grund.

## Checkliste
- Faktisch korrekt?
- Sprachlich klar?
- Strategisch passend?
- Rechtlich/ethisch okay?
- Nächster Schritt eindeutig?

## Regel
Max. ein Review-Zyklus, dann Entscheidung. Kein Endlosschleifen-Editing.
