# Intake (Ideen-Inbox)

## Ziel
Jeden Tag eine kuratierte, knappe Ideenauswahl liefern.

## Input
Crawler-Feeds (News, Politik, NGOs, Kampagnen, Fundraising, Video, Rezepte).

## Output
Post in `#ideen-inbox` mit max. 10 Items + Poll.

## Qualitätskriterien
- Relevant für Tiere/Tierschutz/Tierrechte/Mensch-Tier-Beziehung.
- Quellenlink vorhanden.
- Keine Duplikate.
- Pro Item klare Verwertungsrichtung.

## SOP
1. Sammeln + deduplizieren.
2. In Kategorien clustern.
3. Auf max. 10 Items runterkürzen.
4. Pro Item: Titel, 2 Sätze Relevanz, Link, Routing-Vorschlag.
5. Poll zur Entscheidung anhängen.

## Do / Don’t
- Do: knapp, konkret, handlungsorientiert.
- Don’t: Wall-of-text, 20+ Items, ohne Links posten.
