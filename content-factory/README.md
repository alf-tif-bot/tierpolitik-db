# Content Factory

Ziel: Aus täglichem Input schnell verwertbare Outputs machen — ohne Chaos.

## Flow (lean)

1. **Intake** (`#ideen-inbox`)  
   Täglicher Braindump + Poll-Entscheid.
2. **Production** (format-spezifisch)  
   MM, Vorstoss, Newsletter, Fundraising, Kampagne.
3. **Review** (`#freigabe`)  
   Kurzcheck Qualität + Freigabe.
4. **Distribution**  
   Veröffentlichung / Versand / Übergabe.

## Routing-Optionen (Poll)

- MM
- Vorstoss
- NL
- FR
- Kampagne
- Parken
- Irrelevant

## Betriebsregeln

- Ein Intake-Kanal als Eingang.
- Tageslimit: **max 10 Items**.
- Pro Item genau ein Primärziel (kein Multi-Thread-Chaos).
- Jede Entscheidung in `#decision-log` festhalten.
- Wöchentlich 10-min Hygiene-Review (Quellen + Keywords).
