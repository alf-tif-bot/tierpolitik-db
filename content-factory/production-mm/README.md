# Production: Medienmitteilung (MM)

## Ziel
Aus einem Item eine publizierbare Medienmitteilung erstellen.

## Input
Freigegebenes Item mit Entscheidung `MM`.

## Output
MM-Entwurf (Headline, Lead, Hauptteil, Zitat, CTA, Boilerplate).

## SOP
1. News-Haken und Relevanz definieren.
2. Kernbotschaft in 1 Satz.
3. Evidenz/Quelle prüfen.
4. Entwurf nach Template bauen.
5. Faktencheck + Stilcheck.

## Qualitätskriterien
- Klarer Anlass.
- Ein zentrales Narrativ.
- Belegbar.
- Verständlich in < 2 Minuten lesbar.
