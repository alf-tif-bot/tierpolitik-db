# Production: Newsletter (NL)

## Ziel
Aus Items eine lesbare, aktivierende Newsletter-Ausgabe bauen.

## Input
Freigegebene Items mit Entscheidung `NL`.

## Output
Newsletter-Entwurf (Hook, 3–5 Blöcke, CTA).

## SOP
1. Wochen-/Tagesline festlegen.
2. 3–5 stärkste Items auswählen.
3. Mix aus hard news + leichtem Content sicherstellen.
4. Jeden Block mit Nutzen für Leser:innen formulieren.
5. Mit CTA abschliessen.

## Qualitätskriterien
- Klarer Spannungsbogen.
- Kurz und scannbar.
- Kein Jargon-Overload.
- Eindeutiger Call-to-Action.
