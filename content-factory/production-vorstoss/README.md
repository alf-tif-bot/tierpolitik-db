# Production: Politische Vorstösse

## Ziel
Aus einem Item einen politisch tragfähigen Vorstoss entwickeln.

## Input
Freigegebenes Item mit Entscheidung `Vorstoss`.

## Output
Entwurf für Interpellation / Motion / Postulat (je nach Zweck).

## SOP
1. Zuständigkeit klären (Bund/Kanton/Gemeinde).
2. Vorstosstyp wählen:
   - Interpellation = Auskunft
   - Postulat = Prüfauftrag
   - Motion = verbindlicher Handlungsauftrag
3. Problem + Rechts-/Faktenbasis verdichten.
4. Forderung präzise formulieren.
5. Politisches Timing/Fenster notieren.

## Qualitätskriterien
- Zuständigkeit korrekt.
- Forderung klar umsetzbar.
- Politisch anschlussfähig.
- Begründung belastbar.
