# Production: Fundraising (FR)

## Ziel
Fundraising-Ideen mit NGO-Relevanz in testbare Assets übersetzen.

## Input
Freigegebene Items mit Entscheidung `FR`.

## Output
Kurzbrief mit: Zielgruppe, Offer, Hook, Kanal, Testidee, KPI.

## SOP
1. Zielsegment wählen (Bestand, Neu, Major, Peer-to-peer).
2. Offer definieren (dringend/konkret/wirkungsnah).
3. Story-Hook + Proofpoint formulieren.
4. Kanal bestimmen (Mail, Social, Landing, Event, Kooperation).
5. A/B-Test und KPI festlegen (CR, CPC, Spendenhöhe, Retention).

## Qualitätskriterien
- Emotion + Evidenz im Gleichgewicht.
- Klare Handlungsaufforderung.
- Messbarkeit ab Tag 1.
- Reproduzierbar.
