# Cockpit (Cockpit)

Gemeinsames operatives Cockpit (Tobi + ALF) auf Next.js.

## Start

```bash
cd cockpit
npm install
npm run dev
```

Dann öffnen: http://localhost:3000

## Enthalten
- Shared Board: Open / Doing / Done
- Zuweisung: Tobi / ALF / Beide
- Priorität: high / med / low
- API-Routes in Next.js (`/api/tasks`)
- Local-first Persistenz in `data/db.json`
