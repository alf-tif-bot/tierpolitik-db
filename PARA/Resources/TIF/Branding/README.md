## Operative Design-Regeln für Medienmitteilungen (Working Defaults)

1. Format für Vorschaubilder: **680 × 383 px**, **JPG**
2. Stilpräferenz (laut Tobi):
   - Bevorzugt **pencil sketch / illustrativ** (unaufdringlich, weniger "offensichtlich KI")
   - Alternativ **symbolische, sachliche Motive**
3. Keine reisserischen, hyperrealistischen KI-Looks
4. Motive sollen seriös, klar und thematisch anschlussfähig für TIF sein

## Hinweis
Vor jedem neuen MM-Visual kurz gegen dieses Verzeichnis prüfen (PDF + diese Defaults).
