# Content Factory (PARA/Resources)

Dieses Verzeichnis ist die Obsidian-kompatible Wissensablage für die Content Factory.

## Struktur

- `Quellen/INDEX.md` → zentrale Quellenübersicht
- `Quellen/Quelle - <domain>.md` → eine Note pro Quelle mit Tags + letzten Funden

## Tag-Konvention

- `#content-factory`
- `#quelle/<kategorie>` (z. B. `#quelle/fundraising`)
- `#route/<ziel>` (z. B. `#route/fr`, `#route/vorstoss`)

## Wiki-Links

Quellen sind via `[[Quelle - ...]]` und `[[Quellen-Index (Content Factory)]]` verknüpft.
