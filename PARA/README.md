# PARA Vault

Struktur:
- `Projects/` – zeitlich begrenzte Vorhaben mit klarem Ziel
- `Areas/` – laufende Verantwortungsbereiche (z. B. Gesundheit)
- `Resources/` – Referenzwissen
- `Archive/` – abgeschlossene/pausierte Inhalte

Zusatz:
- `Inbox/` – schnelle Captures
- `Daily/` – Tagesnotizen

## Konvention für Insights
Dateiname:
`YYYY-MM-DD--kurzer-titel.md`

Frontmatter:
```md
---
type: insight
area: health
tags: [health, body]
source: chat
created: 2026-02-22
---
```
